Panduan Setup & Template Database

Pelajari cara menulis rumus, menebalkan teks, membuat paragraf baru, dan memasukkan gambar. Gunakan tabel di bawah untuk menyalin template langsung ke Google Sheets milikmu.

📝 Aturan Penulisan Konten (Wajib Dibaca)

✍️

Format Teks (Bold/Italic)

Jangan menggunakan kode HTML seperti <b>. Sistem ini sudah sangat pintar.

  • Cukup gunakan fitur bawaan Google Sheets.
  • Blok teks lalu tekan Ctrl + B untuk Tebal.
  • Tekan Ctrl + I untuk Miring.
  • Aplikasi akan otomatis mendeteksi gaya teksmu!
↵

Pindah Baris / Paragraf

Kamu tidak perlu mengetik kode <br> untuk memisahkan paragraf.

  • Saat mengetik di sel Sheet, tekan Alt + Enter (atau Ctrl + Enter).
  • Teks akan turun ke baris baru di dalam sel yang sama.
  • Aplikasi akan membaca baris baru tersebut secara otomatis.
🧮

Menulis Rumus

Aplikasi mendukung LaTeX. Gunakan penanda ini di sel Sheet:

  • Rumus Sebaris (Inline):
    Gunakan \( ... \) atau $ ... $
    Contoh: Luas = \(\pi r^2\)
  • Rumus Besar (Block):
    Gunakan $$...$$
    Contoh: $$\frac{1}{2}$$
🖼️

Menyisipkan Gambar

Dua cara memasukkan gambar ke dalam aplikasi:

  • Via Kolom Image_URL: Isi link khusus di kolom ini (berlaku untuk MATERI_IMG).
  • Sisip Tengah Teks (Custom):
    Agar gambar muncul di tengah paragraf/soal, ketik:
    [GAMBAR:https://link.com/gambar.png]
