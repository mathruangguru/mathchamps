Height of Three People
Code : D4-S321

Bar Model Method (Singapore Math)

Use boxes (bars) to compare the height of three people.

Tips: Draw the first bar as a reference. If "taller", draw a longer bar to the right. If "shorter", draw a shorter bar.
Section A: Short Answer Problems with Visual Aids

  • Dayu is 10 cm taller than Siti.
  • Lani is 5 cm shorter than Dayu.

Step 2: What is Lani's height? cm.
2. Edo's height is 135 cm. Beni is 5 cm shorter than Edo. Udin is 8 cm taller than Beni.
(Visualize: Beni's bar is shorter than Edo's, then Udin's bar is longer than Beni's.)
What is Udin's height? cm.
3. Toy Block Tower.
  • Block stack A is 50 cm tall.
  • Block stack B is 2 times the height of stack A.
  • Block stack C is 20 cm shorter than stack B.
What is the height of block stack C? cm.
