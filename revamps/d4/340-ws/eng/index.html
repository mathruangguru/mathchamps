Fractions of a Set
Code : D4-S340
Instructions:

In this worksheet, we will find the number of items belonging to someone if we know the total amount, the amount belonging to someone else, and the fraction value representing a part of the whole set. Use bar models to help you!

Example: Mother has 8 cakes. Brother ate 2 cakes. The remaining cakes belong to Sister, which is \(\frac{3}{4}\) of the total. How many cakes does Sister have?
1 part = 2 cakes. Sister has 3 parts, so 3 × 2 = 6 cakes.
Section A: Calculating with Bar Models

2. There are 6 apples in total. Father ate 1 apple. The rest of the apples were given to Mother, which is \(\frac{5}{6}\) of the total. How many apples does Mother have?
Answer: apples
