COMPARING QUANTITIES

Calculating value based on "how many times as many"

Worked Example

Problem: Ani's number of books is 3 times as many as Budi's number of books. If Budi has 5 books, how many books does Ani have?

Budi
5
Ani
5
5
5
Relationship Analysis
Ani has 3 unit boxes.
Each box has the same value as Budi's.
Calculation
Total = Units × Value
3 × 5 = 15
Answer: Ani has 15 books.
