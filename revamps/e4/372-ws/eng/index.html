WORKSHEET: LINE GRAPH EXPLORATION

Key Concept: Reading Line Graphs
  • Horizontal Axis: Usually shows time (days, months, hours).
  • Vertical Axis: Shows the measured value (temperature, number of books, etc).
  • Upward Line: Value increases. Downward Line: Value decreases.
  • Flat Line: Value stays the same (constant).
