WORKSHEET: DECIMALS

Visual Concept: Shifting Place Value
When we multiply a decimal by 100, each digit shifts 2 places to the left. Imagine the digits jumping to a larger place value column!
Hundreds Tens Ones . Tenths Hundredths
3 . 4 5
← Shift 2 steps to the LEFT (Multiply by 100) ←
3 4 5 . 0 0
Example: 3.45 × 100 = 345
Quick Tip: Move the decimal point 2 steps to the right.
