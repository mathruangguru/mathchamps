WORKSHEET: FRACTIONS AND MONEY

Visual Approach: Finding the Unit Value (Unitary Method)
Use the Bar Model to solve the problem. If the total value is known, find the value of 1 part (unit) first.
Example: Total money 5 units = Rp150.000. So 1 unit = $150.000 \div 5$ = Rp30.000.
Working Logic:

Total (5 units) = Rp150.000

1 unit = $150.000 \div 5$ = Rp30.000

Remaining (3 units) = $3 \times 30.000$

= Rp90.000
