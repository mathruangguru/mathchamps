PANDUAN VISUAL: TARIF BERTAHAP

Konsep: Tarif Berbeda per Jam
Contoh Soal:
Biaya parkir adalah Rp 5.000 untuk 1 jam pertama dan Rp 3.000 untuk setiap jam berikutnya.
Berapa biaya parkir untuk 4 jam?
Langkah Pengerjaan:
  1. Pisahkan jam pertama dengan jam-jam berikutnya.
  2. Jam ke-1 = Rp 5.000.
  3. Sisa waktu = $4 - 1 = 3$ jam.
  4. Biaya sisa waktu = $3 \times \text{Rp } 3.000 = \text{Rp } 9.000$.
  5. Total = $\text{Rp } 5.000 + \text{Rp } 9.000 = \text{Rp } 14.000$.
Tip: Jika waktu parkir adalah "4 jam 15 menit", hitunglah sebagai 5 jam karena setiap "bagian dari jam" biasanya dihitung 1 jam penuh.
