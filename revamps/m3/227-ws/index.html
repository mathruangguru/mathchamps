PERBANDINGAN (RATIO) TIGA BESARAN

Konsep: Menggabungkan Rasio (Link Method)
Untuk menggabungkan dua perbandingan, samakan nilai variabel yang muncul di kedua perbandingan menggunakan KPK (Kelipatan Persekutuan Terkecil).
Contoh: A : B = 2 : 3 dan B : C = 3 : 5
Karena B sama-sama bernilai 3 unit, rasio bisa langsung digabung.
A : B : C = 2 : 3 : 5
Tips Menyamakan (KPK)
Jika A : B = 1 : 2 dan B : C = 3 : 4.
Samakan B (KPK 2 dan 3 adalah 6).
A : B = 1 : 2 → (x3) → 3 : 6
B : C = 3 : 4 → (x2) → 6 : 8
Hasil: 3 : 6 : 8
Gunakan kotak di bawah soal untuk menggambar modelmu sendiri!

3.
