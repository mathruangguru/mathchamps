KUNCI JAWABAN

A. Konversi Pecahan (LO 1)

1. D (60%) 3/5 x 100% = 300/5 % = 60%.
2. B (2,375) 2 + (3 ÷ 8) = 2 + 0,375 = 2,375.
3. A (7/20) Tidak suka = 100% - 65% = 35%. 35/100 = 7/20.
4. A (1,05) 0,75 + 0,5 - 0,2 = 1,25 - 0,2 = 1,05.
5. B (1/4 bagian) Ayah 0,2; Kakak 0,25; Adik 0,3. Total = 0,75. Sisa = 1 - 0,75 = 0,25 = 1/4.

B. Rasio & Bar Model (LO 2)

6. B (25 butir) Merah (3 unit) = 15 -> 1 unit = 5. Biru (5 unit) = 25.
7. C (Rp 88.000) Selisih (7-4)=3 unit = 24.000 -> 1 unit = 8.000. Total 11 unit = 88.000.
8. C (45 cm) Total 8 unit = 120 -> 1 unit = 15. Pendek (3 unit) = 45 cm.
9. C (60 buku) Samakan total unit (KPK 12). Awal 9:3, Akhir 8:4. Beda 1 unit = 5 buku. Total 12 unit = 60.
10. C (Rp 30.000) A:B=8:12, B:C=12:15. A(8u)=16.000 -> 1u=2.000. C(15u)=30.000.
